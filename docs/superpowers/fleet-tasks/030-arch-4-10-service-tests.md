## Instructions

### Pattern to follow

```go
func TestAudiobookService_GetBook_NotFound(t *testing.T) {
    mockStore := new(mocks.MockStore)
    svc := audiobooks.NewService(mockStore)

    mockStore.On("GetBookByID", mock.Anything, "nonexistent").Return(nil, database.ErrNotFound)

    _, err := svc.GetBook(context.Background(), "nonexistent")
    assert.ErrorIs(t, err, database.ErrNotFound)
    mockStore.AssertExpectations(t)
}
```

### Test coverage targets

For each service, write tests covering:
1. **Happy path**: normal input returns expected output
2. **Not found**: store returns ErrNotFound ‚Üí service returns appropriate error
3. **Store error**: store returns generic error ‚Üí service propagates or wraps it
4. **Invalid input**: empty ID, nil params ‚Üí service returns validation error
5. **Idempotency**: where relevant, calling twice is safe

### `AudiobookService` test cases
