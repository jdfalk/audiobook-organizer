# UOS-11 — Migrate Deluge plugin

**Companion human spec:** §11 phase B.

## Branch

```
feat/uos-11-deluge
```

## Goal

Migrate Deluge integration ops:
- `deluge.import-protected-paths` — sync the Deluge-protected paths
  cache
- `deluge.centralization` — copy/reflink files into the central
  library structure (per the existing centralization spec)
- `deluge.path-update` — push updated paths back to Deluge
- Any other Deluge-named scheduled jobs found via
  `grep -rn "deluge" internal/server/scheduler*.go`

## Files to add

1. `internal/plugins/deluge/plugin.go` — plugin shell.
2. `internal/plugins/deluge/protected_paths.go` —
   `protectedPathsSyncDef()`:
   - `ID`: `"deluge.protected-paths-sync"`
   - `Schedule`: every 30min (matching existing cache TTL)
   - `ResumePolicy`: `ResumeRequeue`
3. `internal/plugins/deluge/centralization.go` —
   `centralizationDef()`:
   - `ID`: `"deluge.centralize"`
   - `ResumePolicy`: `ResumeRestart` with checkpoint per file moved
   - `Capabilities`: `[CapLibraryRead, CapLibraryWrite,
     CapFilesRead, CapFilesWrite]`
   - `Cancellable`: `true`
4. `internal/plugins/deluge/path_update.go` —
   `pathUpdateDef()`:
   - `ID`: `"deluge.path-update"`
   - `Triggers`: `[{EventName: "book.relocated"}]`
   - `ResumePolicy`: `ResumeDrop` (each book.relocated re-fires;
     no need to resume)
5. Tests.

## Files to edit

1. `internal/plugins/plugins.go` — import deluge.
2. `internal/server/server.go` — construct + register.
3. `internal/server/scheduler_tasks.go` — remove deluge scheduler
   entries.
4. Existing handlers (find via grep on `deluge` in
   `internal/server/`) become thin redirects.

## Hard rules

- The centralization op MUST use checkpoint-based resume per file
  moved. The existing centralization spec already calls for this;
  do not regress.
- `deluge.path-update` MUST be event-triggered (not direct-called
  from import handlers). Find existing direct callsites and replace
  with `Bus.Publish("book.relocated", ...)`.

## Acceptance criteria

- [ ] `go test ./internal/plugins/deluge/...` passes.
- [ ] `make ci` passes.
- [ ] Manual: trigger centralization on a small subset; restart
      mid-flight; confirm resume picks up where it left off.
